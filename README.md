## Gambling

[I-Link](https://i-link.pro/) Company development
Framework [NestJS](https://docs.nestjs.com/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Documentation

```bash
npx compodoc -p tsconfig.json -s --host 127.0.0.1 --port 8080 --includes additional_documentation
```
